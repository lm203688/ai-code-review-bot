Privacy Policy

Last updated: May 21, 2026

1. Overview

AI Code Review Bot ("the App") is a GitHub App that automatically reviews pull requests. We are committed to protecting your privacy and being transparent about how we handle data.

2. What Data We Access

When you install the App on a repository, it requests the following GitHub permissions:

  • Pull Requests — Read access to PR diffs for code review
  • Contents — Read access to repository files for context
  • Commit Statuses — Write access to post review status checks
  • Issues & Comments — Write access to post review comments on PRs

3. How We Use Data

The App processes your code diffs solely for the purpose of generating code review feedback. Specifically:

  • PR diffs are sent to an AI language model (DeepSeek) for analysis
  • Review results (security findings, quality scores, suggestions) are posted as PR comments
  • No code content is stored beyond the duration of the review request

4. Data Retention

We do not store your source code, PR diffs, or review results on our servers. Code content is processed in real-time and discarded after the review is complete. Review comments posted to GitHub remain under your repository's control.

5. Third-Party Services

The App uses the following third-party service:

  • DeepSeek API — AI model provider for code analysis. DeepSeek processes code diffs temporarily and does not use them for training. See DeepSeek's Privacy Policy.

6. Data Sharing

We do not sell, share, or transfer your data to any third parties beyond what is necessary to provide the review service (i.e., sending diffs to the AI model for analysis).

7. Security

All communications between the App and GitHub use HTTPS encryption. API tokens and webhook secrets are stored securely and never exposed in logs or responses.

8. Your Rights

You can revoke the App's access at any time by uninstalling it from your GitHub repository or organization settings. Upon uninstallation, the App loses all access to your data immediately.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected on this page with an updated revision date.

10. Contact

If you have questions about this Privacy Policy or the App's data practices, please open an issue on our GitHub repository.
